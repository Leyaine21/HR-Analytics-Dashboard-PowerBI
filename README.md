# HR-Analytics-Dashboard-PowerBI
## Overview
This project is an interactive HR Analytics Dashboard built in Microsoft Power BI. The dashboard provides insights into employee demographics, workforce distribution, and attrition trends, helping organizations understand key human resource metrics and make data-driven decisions. The project was created as a learning exercise by following a Power BI tutorial and recreating the dashboard independently to strengthen data visualization and business intelligence skills.

## Key Metrics
The dashboard tracks and visualizes:
- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate (%)
- Average Employee Age
- Average Employee Experience

## Features
### Employee Demographics
- Age group distribution analysis
- Gender-based attrition insights
- Department-wise employee count
  
### Attrition Analysis
- Attrition by department
- Attrition by salary slab
- Attrition trends based on years of experience
- Job role and job satisfaction analysis
  
### Interactive Filtering
- Age-group slicer for dynamic dashboard exploration
- Department selection for focused analysis
